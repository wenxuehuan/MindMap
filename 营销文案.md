# SWDT 思维导图

让想法更清晰，让计划更有序。

SWDT 是一款轻量、高效的 Windows 桌面思维导图应用，专为本地创作、项目规划、知识整理和头脑风暴而设计。无需复杂配置，打开即可开始构建你的思维结构。

## 今日更新：更紧凑、更顺手，也更适合复杂导图

这次更新围绕「看得清、操作快、布局稳」进行了全面优化。从节点编辑、键盘操作到连线布局和大图导航，SWDT 的使用体验再次升级。

### 新增交互式小地图

复杂导图也不再容易迷路。画布右上角新增小地图预览，可以快速掌握整张导图的结构和当前位置。

- 实时显示节点、父子连线、摘要连线和自定义关系线
- 点击小地图即可快速跳转到目标区域
- 支持拖动视口框连续浏览大型导图
- 支持一键开启或关闭，并在当前会话的多个窗口间同步
- 自动适配亮色、暗色和系统主题

### 新增节点展开与收缩

节点较多时，可以通过精致的 `+ / −` 圆形按钮收起暂时不需要查看的分支，让注意力集中在当前内容上。

- 一键隐藏或恢复全部后代节点
- 收缩状态随导图文件保存
- 隐藏节点不会干扰布局、导航、框选和图片导出
- 连线统一汇聚到展开/收缩按钮，结构关系更加清楚

### 键盘操作全面增强

减少鼠标往返，让思路输入更连贯。

- 方向键快速切换到对应方向的节点
- `Space` 或 `F2` 进入节点文字编辑
- `Enter` 添加同级节点
- `Tab` 添加子节点
- `Esc` 退出编辑并保留节点选中状态

节点现在拥有明确的「选中」和「编辑」两种状态，避免误输入或误创建节点。新增节点后会自动进入编辑状态，可以直接开始输入内容。

### 连线与布局能力升级

- 新增尖括号和中括号走线样式
- 连线根据布局方向从节点对应侧面的中点出发
- 修复部分连线从顶部或错误方向出线的问题
- 修复自动布局后摘要节点位置异常的问题
- 添加、删除节点后自动重新整理布局
- 默认节点间距进一步缩小，导图结构更加紧凑
- 只有根节点可以拖动，拖动时整棵子树同步移动，避免结构被意外打乱

### 编辑与保存体验优化

- 移除节点文字的 80 字符限制，长内容可以继续输入并自动换行
- 首次保存时，自动使用第一个根节点的标题作为默认文件名
- 自动处理 Windows 文件名非法字符和保留名称
- 修复工具栏按钮抢走新节点输入焦点的问题

如果你经常使用思维导图整理复杂项目、长篇内容、供应链信息、学习笔记或产品规划，这次更新会让整个编辑过程更加自然、高效。

## 核心亮点

**多文档并行编辑**

同时打开多个思维导图，通过标签页快速切换、重排或分离窗口，让不同项目之间的整理更灵活。

**流畅的画布体验**

支持画布平移、缩放、框选、适配全部内容和居中选中节点。无论导图规模大小，都能轻松浏览和管理。

**强大的节点编辑能力**

快速新增根节点、子节点、同级节点和摘要节点，配合自动布局功能，轻松把零散想法整理成清晰结构。

**丰富的样式配置**

节点颜色、边框、字体、连接线、箭头、线型、画布背景和网格都可以自由调整，让你的导图既实用又美观。

**关系线与摘要节点**

除了传统树状结构，还可以添加自定义连接线，表达跨节点关系；摘要节点则适合归纳重点、整理阶段性结论。

**撤销与重做**

编辑过程更安心，误操作可以快速恢复，适合长时间构建复杂导图。

**本地文件保存**

导图使用 JSON 格式保存，结构清晰，便于备份、迁移和后续扩展。应用还支持最近文件记录，快速回到上次工作内容。

**主题自适应**

支持亮色、暗色和跟随系统主题，适合不同使用环境和个人偏好。

## 适合场景

SWDT 适合用于项目规划、学习笔记、会议整理、产品设计、写作大纲、知识体系搭建以及个人任务拆解。

从一个中心主题开始，把想法一步步展开。SWDT 帮你把混乱的灵感变成清晰的结构。
