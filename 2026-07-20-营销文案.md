# SWDT 重磅更新：思维导图与 Markdown 合二为一，从梳理思路到写成完整内容

先用导图搭结构，再用文档写细节。现在，一个窗口就能完成。

7 月 15 日之后，SWDT 迎来了一次重要升级：在原有思维导图能力之外，新增完整的 Markdown 文档工作区与节点备注系统。无论是整理项目方案、拆解产品需求、记录会议内容，还是构建个人知识库，都可以让“结构”和“内容”真正连接起来。

## 一边画导图，一边写文档

新的双工作区让思维导图与 Markdown 文档并排呈现。左边梳理框架，右边展开细节，不必在多个软件和窗口之间反复切换。

- 思维导图与 Markdown 面板可以同时显示
- 支持拖动交换左右位置，适应不同阅读和写作习惯
- 可自由调整两个面板的宽度比例
- 暂时不需要的面板可以单独关闭，随时从菜单重新打开
- Markdown 目录支持一键展开或收起，长文档定位更高效

从头脑风暴到方案落地，思考过程不再被工具切换打断。

## 不只是纯文本，而是一套完整的 Markdown 编辑体验

SWDT 内置本地 Markdown 编辑器，常用排版能力触手可及：

- 标题、粗体、斜体、删除线与引用
- 左对齐、居中和右对齐
- 无序列表、有序列表与任务列表
- 行内代码、代码块、链接、图片和表格
- 支持亮色、暗色及中英文界面
- 支持撤销、重做以及 `Ctrl+S`、`Ctrl+B`、`Ctrl+I` 等快捷键

编辑器资源随应用一同安装，无需依赖在线 CDN。即使在网络不稳定的环境中，也能保持完整的编辑能力。

## 给每个节点补充一份“深度说明”

导图节点适合表达重点，但复杂内容往往需要更多上下文。现在，每个节点都可以拥有独立的 Markdown 备注。

- 从节点菜单快速添加、编辑或删除备注
- 有备注的节点会显示专属图标，重要信息一眼可见
- 备注窗口可以自由拖动，并从八个方向调整大小
- 备注支持与主文档相同的 Markdown 排版和图片能力
- 所有修改均接入撤销与重做，编辑更安心

任务负责人、参考资料、会议结论、风险说明和执行步骤，都能放在对应节点旁边。导图保持简洁，细节又不会丢失。

## Markdown 文件和图片可以自由进出

已有 Markdown 内容无需重新录入，SWDT 支持直接导入和导出。

- 导入 `.md` 或 `.markdown` 文件并保留原有内容
- 自动识别并嵌入 PNG、JPG、GIF、WebP 等本地图片
- 支持导入 Markdown 中的 Data URI 图片
- 找不到图片时保留原始引用并给出明确提示
- 导出时自动创建配套资源目录，并转换为可迁移的相对路径
- 主文档和节点备注都可以单独导出

内容既可以留在 SWDT 中与导图一起管理，也可以随时导出为通用 Markdown 文件，方便发布、共享或继续加工。

## 一份文件，完整保存结构、正文与素材

新版 SWDT 文件会统一保存：

- 思维导图节点与连接关系
- 完整 Markdown 文档
- 每个节点的 Markdown 备注
- 文档和备注中使用的图片资源

保存时会自动过滤不再使用的图片，避免文件持续膨胀。文件格式升级至版本 2，同时继续兼容旧版导图文件；打开未来版本文件时也会明确提示升级应用，降低数据兼容风险。

## 安装更省心，换一台电脑也能直接使用

新版采用 Windows x64 自包含单文件发布，不要求电脑提前安装 .NET。安装包还会自动检测 Microsoft Edge WebView2 Runtime：系统缺少时自动静默安装，已经具备则直接跳过。

下载、解压、运行安装程序，就可以开始使用。

## 更适合这些使用场景

- **项目规划**：导图拆解目标与任务，Markdown 编写方案和执行细节
- **会议整理**：节点记录议题，备注保存结论、负责人和待办事项
- **产品设计**：导图组织功能架构，文档补充需求、规则与验收标准
- **学习笔记**：知识结构与详细笔记放在同一份文件中
- **写作创作**：先搭大纲，再逐章扩写，结构和正文始终同步可见
- **个人知识库**：用导图建立关联，用 Markdown 沉淀长期内容
